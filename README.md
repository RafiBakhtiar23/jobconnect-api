# JobConnect API

REST API Lowongan Kerja menggunakan Golang, Gin, GORM, JWT, dan MySQL.

## Features

- Register
- Login
- JWT Authentication
- Role Admin & User
- CRUD Lowongan
- Search Lowongan
- Pagination

## Tech Stack

- Golang
- Gin
- GORM
- MySQL
- JWT
- Postman

## Endpoint

POST /register
POST /login

GET /jobs
GET /jobs/:id

POST /jobs
PUT /jobs/:id
DELETE /jobs/:id
